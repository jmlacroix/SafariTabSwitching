Safari Tab Switching by rs

Safari Tab Switching is a Safari SIMBL plugin which allow switching between tabs using ⌘+1…9 like Chrome or Firefox.

Dependencies

  • Safari 5.0 (Snow Leopard)
  • SIMBL 1.9.9

Install

  • Download the installer
  • Run the installer and follow the instructions
  • Restart Safari

Download

You can download the OSX installer with SIMBL library embeded.

You can download this project source code in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/rs/SafariTabSwitching
